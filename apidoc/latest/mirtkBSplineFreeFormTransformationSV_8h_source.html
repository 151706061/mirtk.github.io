34 /**
35  * Free-form transformation parameterized by a stationary velocity field.
36  *
37  * This class implements a free-form transformation which is represented
38  * by a stationary velocity field (2D/3D). The displacement field is obtained
39  * by exponentiating the velocity field, resulting in a diffeomorphic
40  * transformation. A stationary velocity field can be further approximated
41  * from a given displacement field using the group logarithm of diffeomorphisms.
42  */
43 class BSplineFreeFormTransformationSV : public BSplineFreeFormTransformation3D
44 {

288  /// Compute displacement field and derivatives using the scaling and squaring (SS) method
289  ///
290  /// \attention If an input/output displacement field is provided, the resulting

292  ///
293  /// \attention The scaling and squaring cannot be used to obtain the displacements
294  /// generated by a 3D velocity field on a 2D slice only. It always
295  /// must be applied to the entire domain of the velocity field.
296  ///
297  /// \param[in] attr Attributes of output images.
298  /// \param[in,out] d Displacement field generated by stationary velocity field.

302  /// \param[out] dv Partial derivatives of corresponding transformation w.r.t. v.
