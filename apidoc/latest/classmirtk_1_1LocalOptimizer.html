protectedvirtual

Test stopping criteria

The objective function must be up-to-date when this function is called. This is usually the case anyway because the current objective function value after a change of the parameters must be evaluated before this function is called to be able to provide this value as argument.

Note
The objective function value may be NaN in case of a non-parametric deformable surface model. In this case, stopping criteria are based only on the current surface geometry or last node displacements. Stopping criteria based on the objective function value should never be fulfilled in this case and always return false.
