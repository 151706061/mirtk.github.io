35 /**
36  * Base class of volume re-parameterization filters based on discrete linear operators
37  *
38  * The reparameterization is the solution of a system of linear equations.
39  * The output domain of the volumetric map is currently restricted to 3D.
40  * This should not be a practical limitation because these filters are mainly
41  * used to re-parameterize a 3D volume, i.e., a map from a 3D domain to another
42  * 3D domain.
43  */
