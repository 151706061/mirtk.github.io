bisect-dof¶

Synopsis¶

bisect-dof <dofin> <dofout>

Description¶

This command bisects a rigid or affine transformation by calculating the matrix square root of the transformation matrix.

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
