reflect-image¶

Synopsis¶

reflect-image <input> <output> [options]

Description¶

Applies a sequence of one or more spatial reflections to an image. The reflection options are processed in the order given and changing the order can change the result.

Command options¶

-x¶

Reflect x axis.

-y¶

Reflect y axis.

-z¶

Reflect z axis.

-xy, -yx¶

Swap x and y axes.

-xz, -zx¶

Swap x and z axes.

-yz, -zy¶

Swap y and z axes.

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
