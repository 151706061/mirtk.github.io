flip-image¶

Synopsis¶

mirtk flip-image <input> <output> [options]

Description¶

Swaps the two image dimensions at a time, in the order of the input options. Both, the image data and the coordinates of the image origin are swapped each time an input option is processed.

Command options¶

-xy, -yx¶

Swap x and y dimension.

-xz, -zx¶

Swap x and z dimension.

-xt, -tx¶

Swap x and t dimension.

-yz, -zy¶

Swap y and z dimension.

-yt, -ty¶

Swap y and t dimension.

-zt, -tz¶

Swap z and t dimension.

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
