evaluate-distortion¶

Synopsis¶

evaluate-distortion <original> <deformed> <output> [options]

Description¶

Computes the distortion of a surface mesh under deformation and stores the results in the output dataset. If no output dataset is given, the mean and standard deviation are reported only.

Arguments¶

original¶

File name of original dataset.

deformed¶

File name of deformed dataset.

output¶

File name of output dataset.

Command options¶

-areal¶

Compute areal distortion of cells: log_2(area(deformed)/area(original)). (default)

-abs-areal¶

Compute absolute value of -areal distortion.

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
