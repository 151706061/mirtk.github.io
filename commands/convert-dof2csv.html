convert-dof2csv¶

Synopsis¶

mirtk convert-dof2csv <dof> <csv> [options]

Description¶

Converts a transformation to a CSV file which can be loaded into STAR-CCM+ GUI.

Command options¶

-delimiter <str>¶

Delimiting string. (default: ‘ ‘)

-target <image>¶

Lattice points for which to store the displacements. (default: FFD lattice)

-points <file>¶

Point set for which to store the displacements.

-t1 <t>¶

Lower time interval limit. (default: -inf)

-t2 <t>¶

Upper time interval limit. (default: inf)

-dt <t>¶

Temporal resolution. (default: lattice resolution)

Standard options¶

-v, -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h, -help¶

Print help and exit.
