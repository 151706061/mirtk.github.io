evaluate-surface-overlap¶

Synopsis¶

mirtk evaluate-surface-overlap <target> <source> [options]

Description¶

Reports label statistics given two surface meshes and a text file listing the indices of corresponding points. The labels have to be stored as named point data array in both datasets.

Arguments¶

target¶

File name of target surface.

source¶

File name of source surface.

Command options¶

-corr <file>¶

Name of text file listing the point-wise correspondences separated by spaces. (default: NN)

-array¶

<name> Name of array storing labels. (default: labels)

-targetarray <name>¶

Name of target array storing labels. (default: labels)

-sourcearray <name>¶

Name of source array storing labels. (default: labels)

-sep <str>¶

Separating string between printed numbers. (default: ,)

Standard options¶

-v -verbose [n]¶

Increase/Set verbosity of output messages. (default: 0)

-debug [level]¶

Increase/Set debug level for output of intermediate results. (default: 0)

-version [major.minor]¶

Print version and exit or set version to emulate.

-revision¶

Print revision (or version) number only and exit.

-h -help¶

Print help and exit.
