Source Code
===========

The source code of the Medical Image Registration ToolKit (MIRTK) is hosted on `GitHub <https://github.com/>`__
under the `MIRTK Organization <https://github.com/MIRTK/>`__ from which all releases and latest development
versions can be downloaded. See the :doc:`changelog` for a summary of changes in each release.

To download the MIRTK source files using Git_, run

.. code-block:: bash

    git clone --depth=1 -- https://github.com/MIRTK/MIRTK.git
    export MIRTK_SOURCE_DIR="$PWD/MIRTK"
    export MIRTK_BINARY_DIR="$MIRTK_SOURCE_DIR/Build"
    cd $MIRTK_SOURCE_DIR

where the `--depth=1` option requests a shallow clone of the current
development files excluding previous revisions in order to save bandwidth,
download time, and disk space. It may be omitted if you want to see the
development history and contribute to the MIRTK.

The MIRTK repository includes copies of the Boost and Eigen library header files
required to build some of the core modules of the MIRTK. Either install these
libraries using the official installation packages for your operating system
or initialize the desired Git submodules before configuring the MIRTK build:

.. code-block:: bash

    git submodule update --init -- Modules/ThirdParty/Boost
    git submodule update --init -- Modules/ThirdParty/Eigen

The source files of commonly used MIRTK modules are directly included in the
top-level MIRTK project under the Modules/ subdirectory. Further optional
modules which are developed and distributed in their own respective Git
repository are included as Git submodules in the Modules/External/ directory.
To also build these optional modules, initialize and update the needed Git
submodules before (re-)configuring the build files, e.g.:

.. code-block:: bash

    git submodule update --init -- Modules/External/Deformable
    git submodule update --init -- Modules/External/VolumetricMapping

To build the online documentation from the reStructuredText (.rst) files in
the Documentation subdirectory, clone also the MIRTK Documentation repository:

.. code-block:: bash

    git submodule update --init -- Documentation


.. _Git: https://git-scm.com
